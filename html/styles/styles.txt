The HTML Style Attribute
	Setting the style of an html element can be done by using style Attribute
	The HTML style Attribute has the following syntax:
		<tagname style="property:value;">
--
Background color
	The CSS Background color property defines the background color for an HTML element.
				<h1 style="background-color:orange;">This is a Heading</h1>
--
Text color
	The CSS color property defines the text color for an HTML element.
	Example
		<!DOCTYPE html>
		<html>
			<body>
				<h1 style="color:red;">This is a heaing</h1>
				<p style="color:yellow;">This is a Para.</p>
			</body>
		</html>
--
Font family
	The CSS Font-family property defines the Font to be used for an HTML element.
	Example
		<!DOCTYPE html>
		<html>
			<body>
				<h1 style="font-family:italic;">This is a Heading</h1>
				<p style="font-family:algerian;">This is a para.</p>
			</body>
		</html>
--
font Size
	The CSS font-size property defines the text size of an HTML element.
	Example
		<!DOCTYPE html>
		<html>
			<body>
				<h1 style="font-size:40px;">This is a Heading</h1>
				<p style="font-size:10px;">This is a para.</p>
			</body>
		</html>
--
Text Alignment
	The text-align property defines the horizontal alignment for an HTML element.
	Example
		<!DOCTYPE html>
		<html>
			<body>
				<h1 style="text-align:center;">Centered Heading</h1>
				<p style="text-align:right;">Right side paragraph.</p>
			</body>
		</html>
