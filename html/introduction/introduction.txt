What is Hyper Text Markup Language?
What is Language?
	A language is a structured system of communication used by humans.
	Languages can be based on speech and gesture (spoken language), sign, or writing.
	The structure of language is its grammar and the free components are its vocabulary.
--
What is Markup Language?
	Marking up the text within the tags is called markup langauge.
	Example
		HTML, XML, XSLT, PHP, etc...
--
What is Hyper Text?
	Hypertext is a text with hyperlinks, when you click on hyperlinks, it will navigates to other page.
	Hypertext documents are interconnected by hyperlinks, which are typically activated by a mouse click, keypress, or screen touch
--
What is HTML?
	HTML stands for Hyper Text Markup Language.
	It is the only language to create a webpage.
	It describes the structure of a Web page.
	It consist of many elements (approx 100).
	Elements tell the browser how to display the content.
--
What is <!DOCTYPE html>?
	It defines that this document is an HTML5 document.
	It must be written in the top of the page.
	We are giving information to the browser that this document is an html5 document.
	Browser will interpret and execute the html document by following html5 standards.
--

What is an HTML Element?
	HTML element is defined by start tag, some content, and end tag.
	--
	Syntax:
		<tag-name>Some content here</tag-name>
	--
	Example:
		<h1>Content here One</h1>
		<h6>Content here six</h6>
	--
What are Empty Elemnts?
	Some elements have no content.
	These elements are called as empty elements.
	Empty elements have no end tag.
What are the tags that doesn't have end tag?
	<br>, <img>, <link>, <input> etc...
--
What is the browser?
	The purpose of a browser is to read HTML documents and display them correctly.
	Browser does not display the HTML tags, but uses them to determine how to display the document.
	Example
	Chrome, Edge, Firefox, Safari.
--
What is the Server?
	Server is a machine, where we host our web application.
--
What is the difference between web browser and web server?
	web browser
		It will be installed in client machine.
		Example:
			Chrome, Edge, Firefox, Safari.
	Web Server
		It will be installed in server machine.
		Example:
			Tomcat, Jboss, weblogic

Tell me About HTML History?
Since the early days of the World Wide Web, there have been many versions of HTML:
Year	Version
1989	Tim Berners-Lee invented www
1991	Tim Berners-Lee invented HTML
1993	Dave Raggett drafted HTML+
1995	HTML Working Group defined HTML 2.0
1997	W3C Recommendation: HTML 3.2
1999	W3C Recommendation: HTML 4.01
2000	W3C Recommendation: XHTML 1.0
2008	WHATWG HTML5 First Public Draft
2012	WHATWG HTML5 Living Standard
2014	W3C Recommendation: HTML5
2016	W3C Candidate Recommendation: HTML 5.1
2017	W3C Recommendation: HTML5.1 2nd Edition
2017	W3C Recommendation: HTML5.2

What is the difference between html4 and html5?
	Doctype
		HTML4	<!DOCTYPE HTML PUBLIC "-  W3C  DTD HTML 4.01  EN" "http://www.w3.org/TR/html4/strict.dtd">
		HTML5	<!DOCTYPE html>
	New elements
		 <header>, <footer>, <nav>, <Audio>, <video>, <main> etc.
	New form controls
		date and time, email, number, category, title, Url, search, etc.
	Multimedia support
		HTML4	No Support
		HTML5	Audio and Video
	storage
		HTML4	cache and cookies
		HTML5	local storage and session storage
