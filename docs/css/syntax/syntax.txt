					declaration 	declaration

	h1		 	{ color:#000000; text-align:center; }
	
	A CSS consists of a selector and a declaration block.
	Where selector points to the HTML element or Selectors we have in CSS you want to style.
	The declaration block contains one or more declarations separated by semicolons.
	Multiple CSS declaration are saprated with semicolons, and declaration block are surrounded by curly braces.
