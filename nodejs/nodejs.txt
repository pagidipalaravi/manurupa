	Node.js is a JavaScript runtime, Which is built on Chrome's V8 JavaScript engine (JS Compiler)].

	It acts like as a server, it will compile and runs the js files.

What is the difference between interpreter and compiler?

if any error in code, interpreter won't stop the intrepretation into that perticular line.
it will print error in the console then it will intrepret the next line.

if any error in code, compiler will stop there and it will print error in console.
it won't compile next line.
