What is arrow function?
	Anonymous / Nameless function.
	We will use => symbol to write the arrow function.
	Allow us to write smaller function syntax. 
	Will return the value by default.
What is the use of arrow function?
	If we want to write a function within in a function with short syntax and that function cannot be reusable, then use the arrow function.
	Provides you a more accurate way to write the function. 
	Makes your code more readable and structured.
