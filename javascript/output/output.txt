what are the Display Possibilities in javascript?
--
	innerHTML
		To access an HTML element, javascript can use the document.getElementById(id) method.
		This id attribute defines the HTML element and the content inside the HTML element is the innerHTML.
--
	console.log()
		It is used for debugging purpose.
		If we want to display data in console, then write data into console.log() method.
-
	document.write()
		It is used for testing purpose.
		Using document.write() after an HTML document is loaded, it will delete all exixting element.
		The document.write() should be used only for testing.
--
	window.alert()
		We can use an alert box to display data.
		we can skip the window keyword in window.alert, it is an optinal.
--
	window.print()
		javascript does not have any print object or print methods.
		You have to call window.print() method in the browser to print the current page.
		you cannot access output devices from javascript.
