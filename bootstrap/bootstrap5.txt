What is bootstrap?
	Bootstrap is the most popular HTML, CSS, and JavaScript framework for creating responsive, mobile-first websites.
	What is Bootstrap?
	Bootstrap is a free front-end framework for faster and easier web development.
	Bootstrap includes HTML and CSS based design templates for typography, forms, buttons, tables, navigation, modals, image carousels and etc..
	Bootstrap also gives you the ability to easily create responsive designs.
