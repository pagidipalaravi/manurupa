	color-names
	HEX
	RGB, RGBA

How to apply color names?
	In CSS, a color can be specified by predefined color names.
How to apply background color to an element?
	We can apply background by using background-color property.
How to apply border color to an element?
	We can apply the color for borders by using border property.
--
How to apply color in HEX format to element?
	A color can be specified using a hexadecimal form like (#rrggbb), which indicates red, green and blue.
	The hexadecimal values are specified between 00 and ff.
	Sometimes we can write as a three digit code in css like (#f3c).
	it is possible when both the values are same for each components.
==
How to apply color in RGB format to an element?
	In CSS, a color can be specified as an RGB value represents Red, Green and Blue.
	Each parameter red, green and blue defines the intensity of the color between 0 and 255.
How can we get black and white colors by using RGB values ?
	To display black, set all the parameters to 0 like rgb(0, 0, 0).
	To display white, set all the color parameters to 255 like rgb(255, 255, 255).
--
How to apply color in RGBA format to an element?
	RGBA color values are the extention of rgb color values with alpha,	it specifies the opacity of the color.
	which it represents rgba(red, green, blue, alpha)
==
How to apply color in HSL format to an element?
	HSL stands for hue, saturation and lightness.
	Hue defines the intensity of color between 0 and 360.
	Saturation is a percentage value, 0% is gray and 100% is the full color.
	Lightness is also a percentage, 0% is black and 100% is white.
==
How to apply color in HSLA format to an element?
	HSLA color values are an extension of HSL color values with an alpha channel.
	Alpha specifies the opacity for a color.
	alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all)
    Syntax:
	hsla(hue, saturation, lightness, alpha)
